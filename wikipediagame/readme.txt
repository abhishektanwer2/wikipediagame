step 1 :install  all the dependences from requirements.txt
step 2 :type python Wikipedia_game.py "Amitabh bachhan" "kareena kapoor" and press enter

Information about inputs
 input1= title of the wikipedia page which is your start page
 example if you want start  from this link "https://en.wikipedia.org/wiki/Amitabh_Bachchan" than just type "Amitabh Bachchan".

input2 = title of your target page.
example from start  link to target link  "https://en.wikipedia.org/wiki/Kareena_kapoor" than just type "Kareena Kapoor".
